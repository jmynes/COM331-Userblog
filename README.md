# Environment

* Setup Github OAuth (see tabtasks repo)

* Add this to ~/.npmrc

`prefix=~/.npm-global`

* Add this to .zshrc (or bash)

`PATH=$HOME/.npm-global/bin:`

* Append like:
`:$HOME/.npm-global/bin`

* Run this to install Heroku CLI
`npm install heroku-cli -g`
